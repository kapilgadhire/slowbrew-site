# SlowBrew Landing Page

Pre-launch landing page for SlowBrew (slowbrew.ai), an AI-powered dating app built around compatibility-first introductions.

## Stack
Single static HTML file. No build step, no dependencies. Fonts load from Google Fonts.

## Local preview
Open `index.html` in a browser, or run:

```
python3 -m http.server 8000
```

and visit http://localhost:8000

## Deploy
Hosted on GitHub Pages. Any push to `main` updates the live site.
